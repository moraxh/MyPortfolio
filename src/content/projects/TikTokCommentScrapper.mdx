A **TikTok comment scraper** tool designed to extract comments and metadata from TikTok videos efficiently. This fork introduces **asynchronous fetching**, **CSV output support**, and the ability to fetch comments from **multiple URLs or video IDs** simultaneously. Perfect for researchers, data enthusiasts, and trend analysts! 🚀  

## 🚀 Features  

### ⚡ **Enhanced Performance**  
- **Asynchronous Fetching**: Enables faster and more efficient data retrieval.  
- **Multiple URLs/IDs**: Fetch comments from multiple TikTok videos in one go.  

### 📊 **Flexible Output Formats**  
- **JSON Output**: Detailed metadata including captions, usernames, avatars, timestamps, and nested replies.  
- **CSV Support**: Export data in CSV format for easy manipulation and analysis.  

### 🛠️ **User-Friendly Design**  
- **Simple Commands**: Customize data collection with easy-to-use flags and parameters.  
- **Scalable**: Handles multiple videos simultaneously, saving time and effort.  

## 🖼️ Sample Output  

### 💬 **Comment Data**  
The scraper extracts detailed comment data, including:  
- **Captions**: Video captions for context.  
- **Usernames & Avatars**: Commenter details.  
- **Timestamps**: When comments were posted.  
- **Nested Replies**: Full thread of replies for each comment.

## 🛠️ How It Works  

1️⃣ Provide **TikTok video URLs or IDs** as input.  
2️⃣ The scraper fetches comments and metadata **asynchronously** for faster processing.  
3️⃣ Data is organized and exported in **JSON or CSV format**.  
4️⃣ Analyze the output for research, trend analysis, or other purposes.  

## 💡 Why Use This Scraper?  

- **Fast & Scalable**: Asynchronous fetching allows for quick data collection from multiple videos.  
- **Easy to Use**: Simple commands and customizable parameters make it accessible for all users.  
- **Flexible Outputs**: Choose between JSON or CSV formats for your analysis needs.
